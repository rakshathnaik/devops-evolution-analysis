# DevOps Evolution: From Unstructured to Product-Led
This project contains the data and visualization code used in the article.

## Project Structure
- DevOps efficiency.csv: A reconstructed dataset based on the BT Case Study.
- devOps.ipynb: Python code using Matplotlib and Pandas to generate DORA metric graphs.

## How to Reproduce
1. Clone this repo.
2. Ensure you have `pandas` and `matplotlib` installed.
3. Run the script to generate the 3-panel DORA visualization.
